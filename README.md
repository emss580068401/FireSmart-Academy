# 🚒 FireSmart Academy — 消防智多星：職安大挑戰

這是一個專為**新竹縣政府消防局職安科**量身打造的專業「記憶翻卡遊戲」。旨在透過趣味性的圖文配對，強化消防官兵與民眾對於職安知識與消防器材的記憶。

## 🌟 核心特色
- **專業職安圖庫**：支援 30 組自定義職安教學圖片與詳細解說。
- **長輩與孩童友善設計**：
    - **超大字體**：針對銀髮族優化的清晰閱讀介面。
    - **16:9 全屏圖卡**：完整呈現職安器材細節，不裁切。
- **智慧響應式網格**：
    - **電腦版 (橫向)**：自動展開為寬網格，操作流暢。
    - **手機版 (直向)**：自動切換為窄高網格，單手可玩。
- **沉浸式音效系統**：免下載額外檔案，內建清脆翻牌音與過關輕快的和弦回饋。
- **燈箱放大鏡**：點擊任何職安圖片，即可查看清晰的原始大圖。
- **榮譽榜系統**：記錄各難度的通關時間與翻牌次數，激勵學習。

## 🛠 技術棧
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Audio**: Web Audio API (Synthetic SFX)
- **Data**: JSON-to-JS Schema (支援全離線運作)

## 🚀 快速開始
1. 將此儲存庫下載或 Clone 到本機。
2. 直接點擊 `index.html` 即可開始遊戲，不需架設任何伺服器。
3. **自定義教學內容**：修改 `data/cards.js` 檔案，即可輕鬆替換圖片文字。
